# 52-second release feature tour

**Status:** captured, rendered, captioned, and verified from the exact signed
beta.1 portable payload.

## Final sequence

| Time | Picture | Caption |
|---:|---|---|
| 0–6s | Physical Edge in portrait | Your Linux dashboard |
| 6–11s | Live metric widgets | System health at a glance |
| 11–20s | Physical display turns to landscape | Turn the display; EdgeHub follows |
| 20–31s | Manager before and after adding a screen | Add screens live from your desktop |
| 31–39s | Portrait and landscape Manager previews | Design for either orientation |
| 39–47s | Light and matrix theme reflection | Make it yours |
| 47–52s | Hub and Manager end card | 30 widgets, 19 screens, one live Manager |

## Editorial treatment

- 1920x1080 H.264 at 30 fps, 52 seconds, no audio.
- Gentle push-ins, fades, short factual captions, and no rapid flashing.
- English WebVTT captions are supplied separately.
- Real exact-candidate frames only. No synthetic product UI or pointer action is
  presented as live behavior.
- The orientation sequence uses real KScreen rotation and physical-output
  captures. The Manager sequence uses frames before and after verified real
  input that changed the live Hub.
- The end card carries the independent-project disclaimer.
