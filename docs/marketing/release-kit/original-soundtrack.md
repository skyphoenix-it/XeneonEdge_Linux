# Original release soundtrack

The feature tour and Manager theme showcase use an original ambient electronic
soundtrack generated locally by `scripts/render_original_soundtrack.sh`.

The track contains no third-party recording, loop, sample, melody, or downloaded
audio. It is built from sine oscillators, a four-chord progression, a
mathematically generated low percussion pulse, filters, fades, and loudness
normalization. The rendered video carries AAC stereo audio at 48 kHz.
